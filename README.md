# 🏥 Hospital Emergency Room Dashboard – Excel Interactive Analysis
An interactive Excel dashboard built to analyze and visualize Hospital Emergency Room (ER) operations, patient flow, and department performance using dynamic charts, slicers, and KPI cards.
📌 Project Overview
This project focuses on transforming raw hospital ER data into a clean, interactive monthly dashboard that helps stakeholders quickly understand patient admission trends, wait times, satisfaction scores, and department referrals — enabling better resource planning and patient care decisions.
🎯 Objectives

Analyze patient admission status (admitted vs. not admitted)
Study patient distribution across age groups
Measure timeliness — percentage of patients attended within target time
Perform gender-wise patient analysis
Identify departments with the highest patient referrals
Build a month-wise interactive filtering system
